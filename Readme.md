## Local Setup

1. Clone the repository:
   ```
   git clone https://github.com/CharlonTank/abcdef
   ```

2. Install Lamdera: [https://dashboard.lamdera.app/docs/download](https://dashboard.lamdera.app/docs/download)

3. Run the project:
   ```
   lamdera live
   ```
